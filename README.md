# Carbon Care Detailing

**Carbon Care Detailing** is a modern, single-page web application designed to make car detailing services easy to explore and manage. Built with **React**, **TypeScript**, and **Firebase**, the app combines **performance**, **scalability**, and **maintainability** with a sleek, responsive design that looks great on any device.

The platform allows customers to effortlessly browse services while giving administrators a personal dashboard to update content, upload photos, and publish blog posts.

---

## Features

- **Dynamic UI with React** – Build reusable components for fast, interactive user experiences.
- **Tailwind CSS** – Create responsive and visually appealing designs.
- **TypeScript** – Enforce type safety and boost developer productivity.
- **Firebase Backend** – Handle hosting, authentication, and database management.
- **Lightning-Fast Development with Vite** – Quick builds and instant hot reloads.
- **Fully Responsive** – Works perfectly across mobile, tablet, and desktop devices.
- **Code Quality Tools** – ESLint & Prettier for clean, consistent, and maintainable code.

---

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS
- **Backend & Hosting:** Firebase
- **Build Tool:** Vite
- **Code Quality:** ESLint, Prettier
